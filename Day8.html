Head Section (Title, Meta, Link, Script)

1. Title

Title Tag is shown in Web Browser's Title Bar. Between the opening and closing of Title Tag we can give title of our Webpage.

2. Meta Tag

Meta Tag's general definition is Data about data. in which we can give the information about the page.
For Example We have to give Description about the page or keyword related to it. We can give it in meta tag.

Example

  • meta name="Description" content="___Description_About_the_page____"
  • meta charset="UTF-8" (It shows that we are using Character Set of UTF-8 in our Webpage.)
  • meta http-euuiv="refresh" content="2" (This Code will reload our webpage in every 2 Seconds.)
  • meta name="keywords" content="html tag kfm k fdvksk m"

More About Table

In table there are 3 more tag except those we have learnt before.
These are <th>, <thead> , <tbody>

<th>
th means table heading tag. It is used to create table heading. Instead of td we can place th to define that this is heading. This tag Bolds and change the text align to center.
<thead>
It shows that between thead tag there is a heading part of our table. It effects nothing in our webpage but for developer's ease it is written.
<tbody>
Same as thead it effects nothing in our table but it used to shows that what ever data have been typed between this tag; is the body part of our table.
