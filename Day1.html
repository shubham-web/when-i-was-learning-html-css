1. Name of Some Signs

In General there are very common symbols of english which we also use in HTML.

For Example

  • { } These are Curlie Braces.
  • [ ] These are Square Braces.
  • ; This is Semi Colon
  • : This is called Colon
  • / Slash

2. Basics of HTML

HTML is known as Hyper Text Markup Language and it is used for Make the structure of any webpage.

HTML language can be written in any text editor like notepad, notepad++, skylime, Brackets etc. At the time of saving any html document we must have to save it with .html extension.

4. Paragraph Tag

Paraghraph Tag is used to display a pragraph on our webpage.

5. Text Formatting Tags

These tags are used to Design any text in a webpage.

Some Examples are here :-

  • B Tag is for BOLD.
  • I Tag is used for Italic.
  • Mark Tag is to Highlight S.T.
  • del Tag is to line over the Text.
  • Sub tag - H2O
  • Sup tag - 31st ,28th

6. Align Attributes

Align attributes is only used for Heading and Paragraph Tags. it has four values Left, Right, Center and Justify.
