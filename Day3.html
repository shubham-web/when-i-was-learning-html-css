  1. Anchor Tag
  2. Iframe tag

1. Anchor Tag

Anchor tag is used to creat a link on a webpage.

There are three types of links.

  • Local Link
  • Global Link
  • Inter Link

Local Link is a simple offline page or any document's path for example default.html, index.php etc

Global Link is an online link. ex. https://www.facebook.com http://dewaslive.com

Inter Link is a link of the same page but to access the different content of the page. ex. body h1 img #id1 etc.

Anchor tag looks like this.

As shown it has two attributes.
  • href :- The value href is the document which we have to access after clicking the link.
  • target :- By default it's value is "_self" it means the targeted document will open in the same page. it has more values like "_blank" (Open in new window), we can also put the value of iframe names so that the targeted page open in the iframe container.

2. Iframe Tag

Iframe tag has a capablity to create a container on a webpage in which we can view different pages or any document.

Generally it has four attributes:-

  • name :- it's value is user defiened It means any keyword we can write here as a name of that particular iframe
  • src:- the default page which is has to be shown in the iframe container.
  • width:- it defines a width of the square box.
  • height:- it defines a height of the square box.
Note:-
  • Links with https protocol can't be opened in iframe.
