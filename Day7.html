Block & Inline Element

1. Block Element

Who Occupies 100% width of the parent element.

Example

  • Heading Tags (h1, h2, h3, h4, h5, h6).
  • Paragraph Tag (p)
  • Div Tag (Div)
  • Table Tag (Table)
  • Unordered List (ul) etc.

2. Inline Element

Those Element who Occupy only the required space.

Example

  • Image Tag (img)
  • Span Tag (Span)
  • Anchor Tag (a)
  • Input Tag (input) Etc.

Note:Align attr Only works with Block Elements.
