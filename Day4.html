HTML (Types of Protocol and Anchor Tag's attribute)

  1. Types of Protocols
  2. Anchor Tag's attribute "Target"

1. Types of Protocol

There are four types of protocols.

  • http(Hyper text transfer protocol)this is the general protocol which every site have.
  • https(Hyper text transfer protocol secure) it means on site data is being transferred with safety and privately.
  • ftp(File transfer protocol) Torrent sites use this.
  • file Protocol :- to access Computer's file
Note:- In most cases Global link with "https" protocol can't be accessed in iframe.

2. Anchor Tag's attribute "Target".

Traget Attribute have Five Values.

  • _self :- It is the default value of target attribute. It means when we click on the link the hyperlink refrence will open in same tab.
  • _blank :- If we type _blank as the value of target attribute so when we click on the link the page will open in new tab.

    • Note:- "_top" and "_parent" values are only usefull when we have two or more iframes in one webpage

  • _top :- When we need the webpage to be loaded at top level parent.
  • _parent:- When we need the webpage to be loaded at parent parent.
  • Name of any iframe.
