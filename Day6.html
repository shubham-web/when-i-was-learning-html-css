HTML (Table Tag)

  1. Table
    • TR (Table row) Child tag of table.
    • TD (Table data/column) child tag of tr.

1. Table

It is used to Create a table in a webpage.

It has two part first is table row and second is table data.

Attributes

  • boder :- it's value defines the thicness of our table.
  • width :- It's value defines the width of our Table.
  • height :- It's value defines the Height of our Table.
  • Cellpadding :- It's value defines the distance between border and cell data or any content in the cell.
  • cellspacing :- It's value defines the space between two cells of our table.
  • align :- It is used to align our table in webpage.

Table Data Tag's Attributes (td)

  1. colspan :- it is used to merge 2 or more columns in table.
    • For Example if we have to merge first 3 columns of first row in which total columns are 5. For this we have to type Colspan attr in first td and type it's value to 3. and delete Same row's 2 td tags.
  2. rowspan :- it is used to merge 2 or more rows in our table.
    • For Example we have 5 X 5 table and we have to merge 3 rows of it. For this we need to put Rowspan attr in first row's 1st td tag fill it's value to 3. and delete 2nd row's one td tag and 3rd row's 1td tag.
